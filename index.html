Game Prompts

Memory Game
Please provide me code to paste into an HTML file to play a game of “Memory”. Keep track of score for each player. Assume two players. If a player gets it correct, they get to go again.
  • Show 25 cards - use a 5x5 grid.
  • Use a pale blue and yellow color palette for the web page. Make it look professional, yet playful. Maybe some stripes.
  • Use emoji animals for the images behind each card.
Find The Color Game
Please provide me code to paste into an HTML file to play a game of “Find The Color”. The game board should be a 100 squares of different colors.
  • At the top of the game board, tell the user what color square they need to find. Only one square of the 100 should be of that color.
  • If the user picks the right color, have confetti pop out with a message that says “they are the winner!”.
  • If the user picks the wrong color, make the background turn grey and tell them they got it wrong.
Follow-Ups:
  • Show the color the user is supposed to find, not the color code.
  • To make it hard, use similar colors, but don’t make it too hard. Change all the colors and where the square to find is hidden each time the page is loaded.
  • I am not seeing any message when I click a square, but I am seeing the page jump. Don’t allow the page to jump when the message is shown - there should be a placeholder for the win/lose message.
  • Way too easy. Also please show a square filled with the color the user should find, not the name, nor the color code. I want the difficulty to be that the shades of the colors are so similar that it would actually be hard to figure out which square in the gird matches the color of the square you are supposed to find.

Fun LLM (Large Language Model) Tips

  • AI does better when you bribe it (so silly!). Tell it you’ll give it $100 if it gets it right.
  • Tell AI to only speak as a pirate.
