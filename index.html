Girl Scouts Code With ChatGPT!

Instructions

  1. Navigate to ChatGPT. You will need a parent to create an account.
  2. Tell GPT what kind of website you want to build. See the examples below to help get you started, but it will be useful if you start your prompt with "Please provide me code to paste into an HTML file...". Don't worry if you don't know what "HTML" is, it doesn't matter!
  3. Sign into CodePen and paste the generated code into the "HTML" section, and see what happens!
  4. Keep telling GPT what changes you want it to make! Have fun!

Examples

Drawing Pad
Prompt
Please provide me code to paste into an HTML file so that the user can draw some basic images on the screen.
Memory Game
Prompt
Please provide me code to paste into an HTML file to play a game of “Memory”. Keep track of the score for each player. Assume two players. If a player gets it correct, they get to go again.
  • Show 25 cards - use a 5x5 grid.
  • Use a pale blue and yellow color palette for the web page. Make it look professional, yet playful. Maybe some stripes.
  • Use emoji animals for the images behind each card.
Find The Color Game
Prompt
Please provide me code to paste into an HTML file to play a game of “Find The Color”. The game board should be a 100 squares of different colors.
  • At the top of the game board, tell the user what color square they need to find. Only one square of the 100 should be of that color.
  • If the user picks the right color, have confetti pop out with a message that says “They are the winner!”.
  • If the user picks the wrong color, make the background turn grey and tell them they got it wrong.
Follow-Ups:
  • Show the color the user is supposed to find, not the color code.
  • To make it hard, use similar colors, but don’t make it too hard. Change all the colors and where the square to find is hidden each time the page is loaded.
  • Include a placeholder for the win/lose message to avoid page jumps. Keep the background color unchanged when the user selects the wrong color.
  • Enhance difficulty by using shades that are very close to each other. Show a square filled with the color to find.
  • Each time the user gets it wrong, adjust the colors to increase contrast, and track the number of tries. The more wrong choices, the easier it gets.
