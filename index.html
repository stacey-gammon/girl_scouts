Welcome to Girl Scouts Write Web Games with Chat GPT!

Example games plus prompts to get you started!

  • Memory Game
    Please provide me code to paste into an HTML file to play a game of “Memory”. Keep track of score for each player. Assume two players. If a player gets it correct, they get to go again. Show 25 cards - use a 5x5 grid. Use a pale blue and yellow color pallet for the web page. Make it look professional, yet playful. Maybe some stripes. Use emoji animals for the images behind each card.
  • Find me game
    Please provide me code to paste into an HTML file to play a game of “Find me”. The game board should be a 100 squares of different colors. At the top of the game board, tell the user what color square they need to find. Only one square of the 100 should be of that color. If the user picks the right color, play a happy sound and tell them “they are the winner!”. If the user picks the wrong color, play a sad sound and show a sad face.

The limit is your imagination! And AI. Be nice, it might be our overlord some day. :-P
